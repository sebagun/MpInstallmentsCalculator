MercadoPago Installments Calculator

General data of the extension

Author: Sebastián Gun <sebastiangun@gmail.com>
Description: Installments calculator for MercadoPago and MercadoLibre credit card payments.
Versión: 1.0.1
You can grab the source code of this extension in GitHub.

Third parties

This extension uses the MercadoLibre API.
Also uses the jQuery and jquery-jsonp libraries.
Some icons taken from the Open Icon Library.
