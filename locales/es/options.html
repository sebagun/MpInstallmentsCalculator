Calculador de cuotas de MercadoPago

Datos generales de la extensión

Autor: Sebastián Gun <sebastiangun@gmail.com>
Descripción: Calculador de cuotas de MercadoPago y MercadoLibre para pagos con tarjetas de crédito.
Versión:
Puedes obtener el código fuente de esta extensión en GitHub.

Terceras partes

Esta extensión hace uso de la API de MercadoLibre.
También usa la librería jQuery con los plugins jquery-jsonp y tinysort.
Algunos íconos tomados de Open Icon Library.
